Welcome to Hexo!

This is a Guideline of building your own hexo, which is written in English, but easy to understand!

Preparation and Installation

Git Installation

  • Linux/Mac:
sudo apt install git
  • Windows:

Official link: Download git

  • Check whether installed or not:
git --version

Nodejs Installation

  • Linux/Mac:
sudo apt install nodejs
sudo aot install npm
  • Windows:

Official link: nodejs (Suggest you to choose LTS version)

  • Check whether installed or not:
node -v
npm -v

Hexo Installation

  • Linux/Mac:
sudo npm install -g hexo-cli
  • Windows:
npm install -g hexo-cli
  • Check whether installed or not:
hexo -v

Use GitHub to host

Create a new repository on GitHub

â€‹ Firstly, you need to have a GitHub accountâ€¦

â€‹ Then, create a new repository on GitHub and named yourusername.github.io, note that you must named like this (yourusername is your user name), otherwise the deployment cannot be recognized.

Add SSH key to GitHub

â€‹ Back to the terminal, generate the SSH and add it into GitHub so that GitHub can know whether the your computer are visiting the corresponding account.

git cofig --global user.name "yourusername"
git cofig --global user.email "youremail"
ssh-keygen -t rsa -C "youremail"

â€‹ The computer then will tell you the path of .ssh, find it on your computer, and open it via notepad.

â€‹ To say it simpler, SSH is just the ID of your computer, GitHub recognized your computer via this ID. id_rsa is your private key that you cannot show it to others, while id_rsa.pub is the public key, you can copy the info to GitHub.

â€‹ Github -> settings -> personal settings -> SSH and PGP keys -> New SSH key

â€‹ You can verify whether add the SSH key successfully via your terminal:

ssh -T git@github.com

Deploy hexo to GitHub

â€‹ Open your folder, remember the three folders that I let you to concentrate? Now it turns into _config.yml file, open it and add something in the end:

deploy:
  type: git
  repo: https://github.com/yourusername/yourusername.github.io.git
  branch: master

â€‹ *Notice the format must be totally same with me, pay attention to the blanks.

â€‹ Before deployment, you need to install the deploy-git in your computer so that you can deploy your own hexo into GitHub.

sudo npm install hexo-deployer-git --save

â€‹ Finally, deploy your website via the following commands, and you can visit your bolg via http://yourusername.github.io.
